# Portifólio

Um site simples para divulgar meu portifólio.

## Instalação

```shell
npm install
```

## Inicialização

```shell
npm run dev
```

## build

```shell
npm run build
```

```shell
npx http-server ./out -p 3000 -c-1
```

### scan

```shell
npx retire --path ./out
```

```shell
npm audit --omit=dev
```
